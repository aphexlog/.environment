# INIT ENVIRONMENT
To add the environment directives to your shell, add the following to either the ~/.bashrc , ~/.bash_profile , or the ~/profile
  source ~/.enviranment/.vars
  source ~/.enviranment/{path/to/any/other/file-or-script}
